# Loop Engineering

本项目已安装 loop-engineering skill — AI 编程第四层范式。

## 快速开始

设计一个自收敛循环：

```
帮我设计一个 CI 自动修复 loop
```

执行已有契约：

```
运行 .loop/contracts/xxx.yaml 这个 loop
```

审查契约：

```
审查 .loop/contracts/xxx.yaml
```

## 8 个预置模式

| 模式 | 用途 |
|------|------|
| CI Sweeper | 自动修复 CI 失败测试 |
| Bug Fix Loop | 根据测试用例自动修复 bug |
| Code Quality Loop | 扫描修复代码异味 |
| Refactor Loop | 重构 + 行为不变验证 |
| Daily Triage | CI/Issue 每日分类报告 |
| PR Babysitter | 监控 PR CI 状态 |
| Dependency Sweeper | 依赖更新与安全检查 |
| Changelog Drafter | 自动生成变更日志 |
